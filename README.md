# MVP DataOps Docente

Repositorio piloto para preparar un entorno técnico reproducible para soluciones de datos e IA.

## Objetivo
Implementar una arquitectura IA híbrida simple usando:
- Python 3
- Git y GitHub
- Docker
- GitHub Actions
- Render
- Supabase
- PostgreSQL

## Estructura del proyecto
- `app/`: lógica principal de la aplicación
- `tests/`: pruebas
- `data/`: archivos de ejemplo
- `.github/workflows/`: automatización CI/CD
